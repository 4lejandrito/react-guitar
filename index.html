React Guitar

A beautiful and flexible guitar component for React.Hopefully accessible enough to be usable for you.

GitHubStorybook

Its theme describes it as: A typical Spanish guitar with a brown fretboard and golden details.Once you focus on a string you will be able to navigate strings and frets using the arrow keys.When a specific string is focused you can play it by pressing 'p'.
